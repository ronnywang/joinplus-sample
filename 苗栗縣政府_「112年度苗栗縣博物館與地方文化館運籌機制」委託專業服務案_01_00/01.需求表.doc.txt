
                            苗栗縣政府文化觀光局
          「112年度苗栗縣博物館與地方文化館運籌機制」委託專業服務案
                                   需求表
|一、計畫需求|成立苗栗縣博物館與地方文化館發展運籌機制，透過專業團隊的知識與技術 |
|與目標      |協助苗栗縣政府順利完成各項工作，並由區域合作再擴展至全縣互動，以彼 |
|            |此資源結合，創造最大效益，達到各地方文化館資源轉介、進而使得地方之 |
|            |文化能夠永續傳承。                                                 |
|二、計畫項目|文化設施推動辦公室持續運作                                         |
|與內容      |針對各館舍需求專案輔導與建議，並且協助博物館及協作平台計畫執行等。 |
|            |協助辦理文化設施推動委員會及地方文化館各項審查會議(期中報告、期末報|
|            |告、提案審查會議及訪視等)。                                        |
|            |每月報告當月各項計畫進度，並定期向本局開會討論。                   |
|            |駐地經理人會議至少辦理 4 場次：                                    |
|            |邀請專家學者團隊(經理人)協助協作平台或博物館會議、活動串聯、成效評 |
|            |核、館舍展覽、藝文活動規劃，成果報告撰寫、研究調查等。             |
|            |協助撰寫申請中央部會相關博物館及地方文化館之提案計畫。             |
|            |館舍診斷與輔導                                                     |
|            |研擬診斷機制及館所健檢至少 1 次                                    |
|            |盤點歷年補助館舍(含文化部地方文化館第一、二期計畫補助館舍)及想參與 |
|            |地方文化館計畫之館舍，訪視館舍內容須含訪視、輔導紀錄與建議事項(內附|
|            |每一單位訪視輔導照片至少4張）。                                    |
|            |協助辦理建築物公安或消防安全申報、推動環境友善改善評估與設計書圖製 |
|            |作、釐清場域公共服務整備軟體改善需求、公共機能整備硬體改善需求。   |
|            |館所定位輔導及評量機制至少 11 場次                                 |
|            |針對潛力館舍及本計畫獲補助之館舍，全程輔導、追蹤、評估與檢討各計畫 |
|            |案之執行成效，定期向本局匯報，並於期中、期末報告書中呈現工作執行情 |
|            |形報告(含電子檔)，內容須含訪視、輔導紀錄與建議事項(內附每一單位訪視|
|            |輔導照片至少4張）。                                                |
|            |協助館舍研訂區域發展策略，確認本縣館舍應營課題與資源分配原則。     |
|            |依據本縣館舍發展特性，並參考「地方文化館評量暨管考系統」指標項目及 |
|            |「博物館評鑑及認證辦法」規範，辦理112年度評鑑作業，確保各館舍具永續|
|            |營運，且有能力提升自償率。並提出評鑑結果未達指標者，將協助館舍適性 |
|            |發展或轉型與退場等方式，持續活化各館舍。                           |
|            |發展願景共識會議至少 1 場次                                        |
|            |透過共識會議，討論未來各館舍發展及導入周遭館舍、社區等可能性，並討 |
|            |論如何運用創新體驗活動串聯各館舍。                                 |
|            |辦理縣(市)層級館舍經營人才培育及輔導                               |
|            |專家課程：邀請相關領域的專家進行至少 6 小時專業課程。              |
|            |協作工作坊：至少辦理 5場次，媒合有志於文化經營工作的民眾，共同參與 |
|            |或辦理地方館舍活動串聯、館舍展覽、藝文活動規劃及製作等相關館舍發展 |
|            |等議題或活動等。                                                   |
|            |館所交流觀摩：縣外館舍及鄰近文化生活圈相關據點觀摩行程1天活動2梯次 |
|            |，每梯次須設定觀摩主題，參加對象為各地方館舍人員(含各協作平台相關人|
|            |員)，每梯次至少15人為原則。                                        |
|            |跨域加值研究                                                       |
|            |文化知識體驗坊：規劃縣內館舍及鄰近文化生活圈內相關據點體驗行程1天活|
|            |動共 2                                                             |
|            |梯次，媒合文化館與弱勢團體或學校等進行地方特色推廣文化知識體驗坊。 |
|            |資源串聯活動：至少辦理 1                                           |
|            |場次，邀請文化相關工作者或對館舍營運有興趣之青年學子與在地方文化館 |
|            |相互交流及實地參訪，了解博物館或地方文化館推動內容及精神，並產出相 |
|            |關館舍活動方案。                                                   |
|三、工作進度|本委託計畫自決標日起至民國113年1月31日期間內履行採購標的。         |
|及期限      |第一期：廠商應於112年 9 月 15日前提出期中報告3份。                 |
|            |第二期：廠商應於112年 12 月 15日提送期末報告3 份，送機關審查通過後 |
|            |，15日內送成果報告3份。                                            |
|四、驗收及查|本案分二期付款，並於第二期繳交期末報告書經審查通過後，15日內送成果 |
|核          |報告3份及無其他待結案事項後，驗收及查核，付尾款。                  |
|五、預算經費|112年度文教活動－文教創新推動－博物管理－業務費－委辦費            |
|            |新台幣1,150,000元。                                                |
